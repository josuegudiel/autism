# 🧩 Autism — Respaldo en la nube

Este repositorio guarda en la nube todo el contenido de la carpeta
**AUTISMO AGENT** que está en el escritorio de la PC. Así queda respaldado y
accesible desde cualquier lugar.

## 📁 Estructura

```
autism/
└── agente-autismo/     ← aquí van tus archivos
```

## ⬆️ Cómo subir tus archivos (desde la PC, en el navegador)

1. Abre este enlace directo para subir archivos:
   **https://github.com/josuegudiel/autism/upload/main/agente-autismo**
2. Arrastra los archivos de tu carpeta *AUTISMO AGENT* al recuadro.
3. Baja y haz clic en el botón verde **"Commit changes"**.

> 💡 Si tienes subcarpetas, arrastra la carpeta completa: GitHub conserva la
> estructura. Los archivos no pueden pesar más de 25 MB por esta vía.
